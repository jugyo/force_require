force_require
====

It provides the Kernel#force_require to require gems over the bundler.

Install
----

### In Gemfile

    gem "force_require"

### To local

    curl https://github.com/jugyo/force_require/raw/master/lib/force_require.rb > ~/force_require.rb

Usage
----

e.g.

    force_require 'ir_b'

See also `./example`.
